# docs-bussola-premiada

Manual do usuário do **Bússola Premiada**, publicado via GitHub Pages.

O conteúdo é gerado no repositório do plugin (`RIT-DF/bussola-premiada`, pasta
`docs-publico/`) e publicado aqui pelo script `bin/publish-manual.sh`.

Não edite este repositório manualmente — a fonte de verdade é `docs-publico/`.
